Este site foi feito utilizando informações coletadas no site Educador Brasil Escola e com conhecimentos prévios sobre HTML, CSS e Javascript.
O processo foi feito primeiro imaginando como seria o site e rabiscando uma ideia no papel sobre seu design.
Após isso, fui atras de informações no site citado anteriormente e de imagens coerentes com o assunto de cada R.
Para a montagem do site, primeiro foi feito todo o esqueleto em HTML mas com a ideia na mente do que ja tinha sido idealizado no papel de um design. A principio não havia colocado nenhuma informação ou imagem. Apenas montando uma estrutura com os botões que logo seriam estilizados.
Para a estilização com CSS utilizei o site w3schools como auxilio para deixar a pagina o mais próximo do que idealizei. A adição do javascript também foi feita com o auxilio do mesmo site.
Assim que foi constatado que cada botão estava funcionando como eu queria foi a hora de adicionar os textos e imagens. E então, terminei o site.
